隐私政策

最后更新日期: 2025年12月25日

欢迎使用 石墨文档导出工具。我们致力于保护你的隐私并确保你的个人信息安全。本隐私政策说明了你在使用我们的浏览器扩展时，我们关于数据收集和使用的做法。

1. 核心隐私原则

我们相信你的数据属于你。石墨文档导出工具采用本地化架构设计。所有核心功能，包括文件信息的获取和文档导出，都 100% 在你的浏览器本地完成。我们不运行任何存储你个人数据或文档内容的服务器。

2. 数据处理说明

2.1 本地数据（存储在你的设备上）

  • 文档信息： 仅用于导出过程中识别文件。
  • 用户设置： 你的个性化设置（如导出文件夹、导出格式等）。

这些信息使用浏览器的本地存储 API 存储，除非你主动操作，否则永远不会离开你的设备。

2.2 接口调用

本工具会调用石墨文档的官方 API 以获取你的文档列表和导出链接。这些调用均在你的浏览器中以你的身份进行，类似于你亲自在网页上操作。

3. 数据安全

由于所有处理都是本地的，你的数据安全主要受浏览器安全沙箱的保护。我们建议你保持浏览器更新到最新版本，以获得最佳安全性。

4. 第三方服务

石墨文档导出工具不包含任何第三方追踪脚本或分析工具。我们与第三方的互动仅限于：

  • 石墨文档： 仅用于实现文档导出功能。
  • 浏览器商店： Chrome Web Store 提供的标准遥测数据。

5. 你的权利

你对你的数据拥有完全的控制权：

  • 你可以随时在扩展中清除本地数据。
  • 你可以通过卸载扩展立即删除所有本地存储的数据。

6. 联系我们

如果你对本政策有任何疑问，请通过 邮件 与我们联系。
