# 石墨导出插件安装指南
## 一、说明：

1. 该插件为浏览器插件，需要搭配浏览器使用，目前已支持 chrome、firefox、edge 三个主流浏览器。
2. 使用前请先自行安装浏览器，推荐chrome浏览器
3. Tips：chrome浏览器从应用市场安装，需要有科学环境

----
## 二、浏览器插件安装指南-图文版

### 谷歌chrome浏览器
#### 一、离线安装

下载地址：[点击下载](https://github.com/Navyum/chrome-extension-shimo-export/releases/download/v1.0.9/shimo-export-chrome.zip)，先下载离线安装包到电脑。 
备用下载地址：[夸克备用](https://pan.quark.cn/s/98224746a524) 


安装步骤：
1. 在地址栏输入 chrome://extensions 快速进入
   <p align="center"><img src="https://cdn.jsdelivr.net/gh/Navyum/imgbed@pic/IMG/3b579258d456470c56ca636e336fbd04.png" width="80%"></p>
2. 打开开发者模式
   <p align="center"><img src="https://cdn.jsdelivr.net/gh/Navyum/imgbed@pic/IMG/7fa12a0f6bb115e5eedacaf77e280094.png" width="80%"></p>
3. 打开下载的压缩包目录，无需解压，直接拖找到扩展程序页面
   <p align="center"><img src="https://cdn.jsdelivr.net/gh/Navyum/imgbed@pic/IMG/007738a8f703ea45287603a78db46e0a.png" width="80%"></p>
4. 确认是否安装成功，确认插件开启
   <p align="center"><img src="https://cdn.jsdelivr.net/gh/Navyum/imgbed@pic/IMG/f179e4ea0df8ee7b883f1ec6ebc2c9e8.png" width="80%"></p>
5. 将插件固定到工具栏
   <p align="center"><img src="https://cdn.jsdelivr.net/gh/Navyum/imgbed@pic/IMG/3f2357c72eb22fbc441485f5bab3d72e.png" width="80%"></p>
6. 点击插件进行使用
   <p align="center"><img src="https://cdn.jsdelivr.net/gh/Navyum/imgbed@pic/IMG/f807985613fb5fabe886daf11632a618.png" width="80%"></p>



#### 二、chrome应用市场安装  👍👍👍
注意⚠️：需要拥有科学的网络环境

chrome应用市场下载地址：[点击直达](https://chromewebstore.google.com/detail/jdipfhjpijkdjbefbaehnimligdldhdp?utm_source=xianyu)

1. 打开上方直达应用市场地址，或者搜索“石墨导出”，点击添加插件
<p align="center"><img src="https://cdn.jsdelivr.net/gh/Navyum/imgbed@pic/IMG/b54dd84dcb7e524d6bd15af4a1183c6b.png" width="80%"></p>

2. 点击固定到状态栏，方便后续使用
<p align="center"><img src="https://cdn.jsdelivr.net/gh/Navyum/imgbed@pic/IMG/55f9fb25ff7aa00d7e3f07df34001122.png" width="80%"></p>

3. 点击插件进行使用
<p align="center"><img src="https://cdn.jsdelivr.net/gh/Navyum/imgbed@pic/IMG/55f9fb25ff7aa00d7e3f07df34001122.png" width="80%"></p>
